### 纠正上次的错误判断

我之前说"被踢到淘宝登录页"是错的。刚才我查了 server/data/logs/ 里实际的两张 error 截图（product_2_1781840121517_error.png 和 product_2_1781839977236_error.png），页面内容清清楚楚显示是**千牛商品管理后台的 in_sale 商品列表页**，左侧有商品管理、仓储库存等菜单，右侧是商品列表（有梯子、床笠、手机支架等）。

`json

  "step": "error",
  "detail": {
    "error": "CATEGORY_CAT_IDS is not defined"
  }

代码逻辑是：如果分类名不在硬编码的 catId 映射表里 → 走 AI 搜索；如果在表里 → 直接跳转到 publish.htm?catId=...。但因为 CATEGORY_CAT_IDS 没定义，即使 cat="花茶" 这样明确在表中的分类，也会在第一步就报 ReferenceError 挂掉。

### 需要你修

在 server/src/services/taobao-auto-list.js 中加上映射表，比如：

`javascript
const CATEGORY_CAT_IDS = {
  '花茶': 125242010,

  '茶叶': 125242010,
  // 其他常用分类...

`
