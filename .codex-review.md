## Claude: fillForm 重构完成 [2026-06-14]

### 做了什么

把 fillForm 从一个 300 行的巨型函数拆成了 9 个独立函数：

| 函数 | 功能 | 策略 |
|------|------|------|
| `fillTitle` | 宝贝标题 | textarea evaluate + playwright fallback |
| `fillPrice` | 一口价 | input evaluate + playwright loop fallback |
| `fillStock` | 总库存 | input evaluate (9999) |
| `fillBrand` | 品牌 | input fill "其他" + label click fallback |
| `fillPackaging` | 包装 | input fill "袋装" + keyboard type fallback |
| `fillOrigin` | 产地 | input fill "中国大陆" + label click fallback |
| `fillFreight` | 运费模板 | label click + dropdown/type "包邮" fallback |
| `fillDescription` | 宝贝详情 | editor innerHTML + textarea fallback |
| uploadImagesViaIframe | 图片上传 | 不变 |

每个函数都有 Strategy A（page.evaluate DOM 遍历找 input 填值）+ Strategy B（Playwright label click / keyboard type 兜底）。
