## Latest changes (Claude, 2026-06-19)

### Changes made
1. **Profile persistence**: Added `copyProfileBack()` — after browser close, Playwright's changes (login cookies) are synced from the copy back to `taobao-profile` dir. Previously login state was lost after every run.
2. **Context close**: Added `context.close()` at end of `batchListToTaobao()` to flush all browser state before syncing profile back.
3. **Background task runner**: Split the auto-list into async task system:
   - POST `/api/listings/auto-list` returns `{ taskId }` immediately
   - Frontend polls `GET /api/listings/auto-list-task/:taskId` every 2s
   - Live progress text visible in UI
   - Cancel support: `POST /api/listings/auto-list-task/:taskId/cancel`
4. **Progress callbacks**: `batchListToTaobao` now accepts `options.onProgress` — key steps report progress text.
5. **Frontend update**: ListingManager shows live progress card with cancel button.
6. **Codex path**: Fixed project location in this file.
